Font size appearing abnormally small on smaller screens?

Check that you have inserted the viewport metadata option in a meta tag!

<meta name="viewport" content="width=device-width, initial-scale=1.0"/>

gap grid/flexbox property

When trying to provide space around elements in flexbox and grid you might have done this:

.child-element {
    margin: .5em;
}

This can provide unncessary space to left and right of the elements in the first and last rows you may not want

Instead you can use the gap property to provide space in the inside only: 

.parent-element {
    gap: .5em;
}

Why do websites use text-shadow with small properties?

Automatic scroll to close to element?

When you have a link that points to an element in your page you may run into the trouble of the scroll appearing too close to the element content or border.

No counteract this simply add padding to the top of the element:

html {
    ...
    scroll-behavior: smooth;
    ...
}
:is(h2, h3, h4, h5, h6) {
    padding-top: 2em;
    /* stop scroll 2em above the element content */
}

position:sticky not working?

Check for ancestor elements that have overflow: hidden because that may be the culprit.

position: sticky will not work if any of its ancestor elements have overflow: hidden, overflow: scroll, or overflow: auto. This is because sticky positioning is based on the parent elementâ€™s scroll container, so if the parent has overflow settings that prevent scrolling, sticky behavior is disabled.

Also make sure to provide at least one of the top, right, bottom, or left properties.

    Using insertBefore(undefined, ...)

    The method insertBefore() expects two arguments: the node to insert and the reference node. If you pass undefined as the reference node, it essentially appends the new node to the end of the parent.

    MDN Spec on insertBefore:

    If the referenceNode is null or not found, the new node is inserted at the end of the parent.
