The Skeleton

To start this To-Do List application we must first create the skeleton of the HTML page. It contains your typical:

  • doctype
  • html root
  • head / metadata
  • body
  • script element
<!doctype html>
<html lang="en">
    <head>
        <meta charset="utf-8"/>
        <meta name="viewport" content="width=device-width, initial-scale=1.0">

        <title>To-Do List</title>

        <link rel="stylesheet" href="index.css"/>
    </head>
    <body>
        <!-- Web page content to be displayed on browser-->
        
        <script>
            // javascript instructions
        </script>
    </body>
</html>

Breaking it down

Document Type Declaration

In almost every HTML file you create you will need a document type declaration or doctype to indicate the HTML version you are using in the document. In our case, we want to use the latest and greatest version of HTML -- HTML 5 -- so we simply type:

<!doctype html>
<!-- You will also encounter it typed out like: -->
<!DOCTYPE html>
<!-- Both are the same because the casing is ignored! -->

Root Element

Then, after letting the browser know what version of HTML we are using, we want to create the element in which our web page will go in. The html element 

<html>
    ...
</html>

Metadata Elements

The Elements of a HTML page can be separated into two parts:

  • head
  • body

The head elements contains metadata elements or information that describes the web page. For starters, and simplest to understand, is the title element that gives your webpage a title in the browser tab. A more advanced tag is the meta tag that can be used to make your website appear in a search engine through attributes like:

<meta name="description" content="Simple ToDo List">
<meta charset="utf-8"/>
<meta name="viewport" content="width=device-width, initial-scale=1.0">

External Stylesheet

It's good practice to put your css styles in a separate file to reduce clutter. In order to apply those styles to the current web page you have to link them together by using the link tag.

<link href="index.css" rel="stylsheet"/>

Body

As mentioned earlier, an HTML page can be divided into two parts: head and body. The body element contains the content to be displayed on a web browser.

<body>
    ...
</body>

JavaScript

JavaScript can be added to your web page by directly embedding it in the body or providing a link to an external file.

<script>// javacsript code directly in body</script>
<script src="index.js"></script>
